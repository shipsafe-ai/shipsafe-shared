## Devpost description (≈145 words)
AgentOps is fleet observability for AI agents — the capstone that watches the watchers. Once you deploy autonomous agents, they become a new black box: when one slows down, its latency cascades across every agent that depends on it, and at 3am you can't tell which one started it. AgentOps ingests the fleet's OpenTelemetry traces into Dynatrace and queries them back through the Dynatrace MCP server with Grail DQL — computing per-agent p99 latency, detecting cross-agent cascades (one distributed trace with errors in multiple services), and summing real Gemini token spend per agent and model. Gemini on Vertex AI then synthesizes a causal fleet postmortem with captured chain-of-thought, an adversarial Critic checks it, and every verdict ends at human review. Demonstrated on the ShipSafe fleet during a simulated crisis; works for any AI agents on Cloud Run that emit OpenTelemetry.
